A comprehensive data visualization project built with Power BI that explores the financial performance, critical reception, and release trends of Pixar Animation Studios' filmography.

Project Overview
This project provides a deep dive into Pixar's movie history, analyzing how their films have performed at the global box office relative to their budgets. It tracks key performance indicators (KPIs) to identify trends in genres and audience scores over nearly three decades.

Key Metrics Tracked
Total Profit: 10bn 
Total Box Office: 13bn 
Total Budget: 3bn 
Average Critical Score: 81.85 
Average Runtime: 100.60 minutes 

Features
Financial Performance Ranking: Visualizes the "Top 10 Performing" and "Top 10 Profitable" movies globally, led by hits like Incredibles 2 and Toy Story 3.
Genre Distribution: A breakdown showing the dominance of different genres, with Adventure making up 80% of the catalog in the filtered view.
Time-Series Analysis: A "Releases By Year" chart tracking Pixar's production output from 1995 to the late 2010s.
Interactive Slicers: Users can filter the entire dashboard by Specific Film Title, Genre, Release Date Range, and Film Rating.

Tech Stack
Tool: Power BI Desktop 
Data Modeling: DAX (Data Analysis Expressions) for custom KPI measures.
Visualization: Bar charts, line graphs, pie charts, and card visual# Pixar-Movies-Analysis-Dashboard
